Web de Registro de Usuarios

La Web de Registro de Usuarios de Discord Bot es una aplicación web diseñada para permitir a los usuarios registrarse y compartir información sobre ellos mismos, incluyendo sus perfiles de redes sociales, información de contacto y una breve descripción personal. El registro se realiza a través de la plataforma de autenticación de Discord, lo que permite a los usuarios iniciar sesión en la aplicación de forma segura y rápida.

Instrucciones de configuración

  1. Descargue el código fuente de la aplicación y colóquelo en un servidor web que admita PHP.
  2. Cree una nueva aplicación Discord en el portal de desarrolladores de Discord.
  3. Copie el Client ID y el Client Secret de la aplicación Discord y péguelos en los campos correspondientes en el script del html.
  4. Configure la URL de redireccionamiento en la sección "OAuth2" de la página de configuración de la aplicación Discord. La URL debe ser la dirección de la página web donde se aloja la aplicación, seguida de /form/form.php.
  5. Guarde los cambios y cargue la página de registro de usuarios en su navegador web.

Creando el perfil:

Para crear el perfil en la aplicación, siga los siguientes pasos:

  1. Inicie sesión en la plataforma de autenticación de Discord haciendo clic en el botón "Discord Login" en la página de registro de usuarios.
  2. Seleccione su grado y curso.
  3. Haga clic en el botón "Siguiente" para continuar con el registro.
  4. Proporcione información adicional sobre usted, como sus perfiles de redes sociales, información de contacto y una breve descripción personal.
  5. Una vez que haya completado su perfil, haga clic en el botón "Guardar" para guardar su información.

Preguntas frecuentes


¿Cómo puedo actualizar mi perfil?

Para actualizar su perfil, debe volver a iniciar sesión en la plataforma de autenticación de Discord y volver a completar el formulario de registro.

¿Cómo puedo eliminar mi cuenta?

Para eliminar su cuenta, contacte desde su cuenta de Discord con el administrador en el que se está usando el Bot.

¿Cómo puedo contactar al soporte técnico?

Para contactar al soporte técnico, envíe un correo electrónico a app360.sistemas@gmail.com.

Descripción del JavaScript

El JavaScript contiene código que maneja la interacción del usuario con el formulario de registro y la plataforma de autenticación de Discord. El código se divide en dos funciones principales:

  • discordLogin(): esta función se activa cuando se hace clic en el botón "Discord Login" en la página de registro de usuarios. Abre una ventana emergente que muestra la página de autenticación de Discord, lo que permite al usuario iniciar sesión de forma segura.
  • exchangeAuthorizationCodeForToken(code): esta función se llama después de que el usuario inicia sesión en la plataforma de autenticación de Discord y se le concede un código de autorización. Hace una solicitud a la API de Discord para intercambiar el código de autorización por un token de acceso, que se utiliza para completar el registro de usuario.

Descripción del archivo PHP

El archivo PHP form.php procesa los datos del formulario de registro y los almacena en una base de datos. También genera el perfil para el usuario y lo envía a la API a través de una solicitud HTTP POST. El archivo se divide en tres secciones principales:

  • Recuperación de datos: el archivo PHP recupera los datos enviados desde el formulario de registro y los almacena en variables.
  • Validación de datos: el archivo PHP valida los datos del formulario para asegurarse de que se hayan proporcionado todos los campos requeridos y que se hayan ingresado datos válidos.
  • Almacenamiento de datos: el archivo PHP almacena los datos del formulario en una base de datos. Luego, envía la información de acceso a la API a través de una solicitud HTTP POST.

Conclusión

La Web de Registro de Usuarios de Discord Bot es una aplicación web útil que permite a los usuarios registrarse y compartir información sobre ellos mismos en un servidor de Discord. La plataforma de autenticación de Discord garantiza que el registro sea seguro y rápido, y el código fuente de la aplicación es fácilmente personalizable para adaptarse a las necesidades de cualquier proyecto. Con la información proporcionada en esta documentación, los usuarios pueden configurar y utilizar la aplicación con facilidad.



Documentación del Bot de Discord

Descripción General del Bot

Este bot es una aplicación de Discord que se ejecuta en Python y utiliza la biblioteca nextcord. El bot está diseñado para interactuar con los usuarios de Discord y proporcionar información sobre los miembros registrados en la aplicación usuariozombie.com.

Instrucciones de Instalación

Para instalar y ejecutar el bot, siga estos pasos:

  1. Clone o descargue el código del repositorio de GitHub en su máquina local.
  2. Instale las dependencias necesarias ejecutando el comando pip install -r requirements.txt en la línea de comando.
  3. Cree un archivo config.json y agregue su token de bot de Discord, su token de API de usuariozombie.com, el prefijo del comando que desea utilizar y el ID del canal de bienvenida en su servidor de Discord. También puede agregar una lista blanca de IDs de usuario que tendrán acceso a los comandos de administrador.
  4. Ejecute el archivo main.py en la línea de comando para iniciar el bot.

Configuración del Bot

Antes de ejecutar el bot, debe configurar algunas opciones en el archivo config.json. Aquí hay una descripción de cada una de las opciones:

  • BotToken: El token de bot de Discord. Puede obtenerlo en la página de Desarrolladores de Discord.
  • APIToken: El token de API de usuariozombie.com. Puede obtenerlo en la página de usuariozombie.com.
  • Prefix: El prefijo que desea utilizar para los comandos del bot.
  • WelcomeChannel: El ID del canal de bienvenida en su servidor de Discord.
  • Whitelist: Una lista blanca de IDs de usuario que tendrán acceso a los comandos de administrador.
  • Permissions: Las permisos de Discord que desea conceder a su bot. Puede obtener el número de permisos necesario en la página de Desarrolladores de Discord.

Comandos

El bot tiene varios comandos que se pueden utilizar en Discord. Aquí hay una descripción de cada uno de ellos:

  • checkprofile: Verifica el perfil de un usuario en usuariozombie.com. Si no se menciona a un usuario, se verifica el perfil del usuario que ejecuta el comando. El comando se puede utilizar en Discord a través de /profile checkprofile [usuario].
  • help: Muestra una lista de los comandos disponibles y su descripción. El comando se puede utilizar en Discord a través de /profile help.
  • load: Carga el módulo commands. Solo los usuarios en la lista blanca pueden utilizar este comando. El comando se puede utilizar en Discord a través de !load.
  • reload: Recarga el módulo commands. Solo los usuarios en la lista blanca pueden utilizar este comando. El comando se puede utilizar en Discord a través de !reload.
  • unload: Descarga el módulo commands. Solo los usuarios en la lista blanca pueden utilizar este comando. El comando se puede utilizar en Discord a través de !unload.

Personalización del Bot

Si desea personalizar el bot, puede agregar más comandos o cambiar los que ya existen. Para agregar un nuevo comando, debe seguir los siguientes pasos:

  • Agregue una nueva entrada al diccionario de comandos en el archivo "commands.py" con el nombre del comando y la función correspondiente que se debe ejecutar.
  • Actualice el comando "help" con una descripción del nuevo comando.
  • También puede personalizar el mensaje de bienvenida que el bot envía cuando un nuevo usuario se une al servidor. Para hacerlo, siga estos pasos:
    1. Abra el archivo "commands.py"
    2. Busque la función "on_member_join" y modifique el mensaje de bienvenida según sus preferencias.
    3. Además, puede personalizar la apariencia del bot, como su nombre y su avatar, en la configuración del bot en la página de Desarrolladores de Discord.

Conclusión

Conclusión Este bot es una aplicación útil para cualquier servidor de Discord ya que permite a los usuarios verificar fácilmente los perfiles de otros miembros registrados además de que brinda privacidad ya que solo los usuarios que estén dentro del servidor podrán acceder a la información de la persona que ella quiera dar, siendo más privado que teniéndolo en tu perfil de discord ya que esto permite que cualquier usuario pueda acceder a ella. Además brinda a los administradores del servidor el control total sobre quién puede utilizar los comandos del bot. Además, es altamente personalizable, lo que permite a los administradores agregar nuevos comandos o modificar los existentes según sea necesario. Si desea utilizar este bot en su servidor de Discord, siga las instrucciones de instalación y personalización proporcionadas en esta documentación.



Documentación de la API


Introducción POST GET-MEMBER GET-ALL-MEMBERS

API:

La API de este proyecto tiene como objetivo registrar y obtener información de los miembros de una organización. La API es accesible a través de HTTP y está basada en Flask, un framework de Python. Se debe proporcionar un token para acceder a la API, que se encuentra en el archivo de configuración config.json.

URL base:

La URL base de la API es http://localhost:4000/.

POST /post

Este recurso permite agregar un nuevo miembro a la organización.

Parámetros:
Nombre Tipo Descripción
discord string Nombre de usuario de Discord del nuevo miembro.
email string Dirección de correo electrónico del nuevo miembro.
curso string Curso actual del nuevo miembro.
year string Año actual del nuevo miembro.
github string Nombre de usuario de GitHub del nuevo miembro.
twitter string Nombre de usuario de Twitter del nuevo miembro.
instagram string Nombre de usuario de Instagram del nuevo miembro.
fname string Nombre del nuevo miembro.
lname string Apellido del nuevo miembro.
phone string Número de teléfono del nuevo miembro.
description string Descripción del nuevo miembro.
Respuestas:
  • 200 OK en caso de éxito.
  • 498 Token inválido en caso de que el token proporcionado no sea válido.

GET /members/{userid}

Este recurso permite obtener información de un miembro específico de la organización.

Parámetros

Nombre Tipo Descripción
userid int ID del miembro a buscar.

Respuestas

Código Descripción
200 OK en caso de éxito.
404 Not Found en caso de que el miembro no sea encontrado.
498 Token invalido en caso de que el token proporcionado no sea válido.

GET /allmembers

Este recurso permite obtener información de todos los miembros de la organización.

Respuestas

Código Descripción
200 OK en caso de éxito.
498 Token invalido en caso de que el token proporcionado no sea válido.
